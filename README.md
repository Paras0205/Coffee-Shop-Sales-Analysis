# ☕ Coffee Shop Sales Analysis Dashboard

## 📊 Project Overview

This project focuses on analyzing Coffee Shop sales data to uncover valuable business insights related to sales performance, customer purchasing behavior, product demand, and store performance.

Using SQL for data analysis and Microsoft Excel for dashboard creation, this project transforms raw transactional data into an interactive dashboard that helps stakeholders make data-driven business decisions.

---

## 🎯 Objective

The goal of this project is to answer key business questions such as:

- Which products generate the highest sales?
- Which store location performs the best?
- What are the peak customer ordering hours?
- Which product categories contribute the most revenue?
- How does customer purchasing behavior vary across weekdays?
- What are the most popular product sizes?

## 🔍 Business Insights

### 📌 Customer Behavior

- Most customer purchases occur during morning hours.
- Customer activity gradually decreases later in the day.

### 📌 Revenue Insights

- Coffee contributes the highest share of total revenue.
- Tea products also contribute significantly to overall sales.

### 📌 Store Performance

- Astoria generated the highest revenue among all locations.
- Store performance remains relatively balanced across locations.

### 📌 Product Performance

Top-selling products include:

- Barista Espresso
- Brewed Black Tea
- Brewed Chai Tea
- Gourmet Brewed Coffee
- Hot Chocolate

### 📌 Product Size Trends

- Large and Regular sizes account for the majority of orders.
